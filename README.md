# caderno-tematico-notebooklm
Caderno temático desenvolvido com o NotebookLM para estudos de SQL utilizando IA como ferramenta de aprendizagem
🔎Contexto e Objetivos

Este projeto foi desenvolvido como parte do desafio da DIO utilizando o NotebookLM como ferramenta de aprendizagem esse tema escolhido foi SQL (Structured Query Language), por ser um dos conteúdos fundamentais para estudantes e para profissionais da área de dados e desenvolvimento
🎲Objetivos
1.Aprender os conceitos básicos de SQL.
2.Entender a diferença entre comandos DDL, DML e DQL.
3.Revisar comandos como INSERT, UPDATE, SELECT e DELETE.
4.Conhecer recursos avançados como JOIN, Window Functions, Views, Índices e ETL.
5.Utilizar a Inteligência Artificial para organizar estudos e produzir resumos.

🎲Resultado
Recebi uma explicação sobre SQL como linguagem padrão utilizada para consultar, armazenar e manipular dados em bancos de dados relacionais, além da divisão entre DDL, DML e DQL.
Prompt 2
❓Pergunta

🎲Resultado

UPDATE altera registros já existentes.

⚠️Dificuldades Encontradas
Durante os testes observei que algumas respostas ficaram muito técnicas, principalmente quando tratavam de recursos avançados como HeatWave, AutoML e Window Functions.
Para melhorar os resultados, utilizei perguntas mais específicas e solicitei explicações em linguagem mais simples, o que tornou o conteúdo mais fácil de compreender.
📚 Miniguia de Estudos
SQL: linguagem usada para consultar e manipular bancos de dados.
DDL: cria e altera tabelas (CREATE, ALTER, DROP).
DML: insere, atualiza e remove dados (INSERT, UPDATE, DELETE).
DQL: consulta informações (SELECT).
JOIN: relaciona dados de tabelas diferentes.
📖 Glossário
SQL: linguagem para bancos de dados.
Tabela: onde os dados são armazenados.
Registro: uma linha da tabela.
JOIN: une dados de tabelas diferentes.
SELECT: consulta informações.
